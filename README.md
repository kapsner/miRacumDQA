# docker-shiny_baser

- this is a basic Shiny container  

- put your Shiny app to the folder "shiny_app"

- add your required R packages to the Dockerfile before building the image 

- to build your image:
```
cd ./docker/
chmod +x build_image.sh
./build_image.sh
```
